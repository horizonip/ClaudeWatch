# Changelog

All notable changes to Claude Usage Widget will be documented in this file.

## [1.0.0] - 2026-02-04

### Added
- Initial release
- Display 5-hour rolling usage percentage
- Display 7-day usage percentage
- Color-coded progress bars (green/yellow/red)
- Smart refresh based on usage level
- Desktop widget with edge snapping
- Session cookie authentication
- Always-on-top toggle
- Reset time countdown display
- Demo mode for testing (`--demo` flag)
- Configuration via INI file
- Debug response logging for troubleshooting
